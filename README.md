# Legal Information Extraction & Case Summarization System

## 📌 Overview

This project is an AI-driven NLP system designed to extract legal information and generate concise case summaries.

## 🚀 Features

* Legal text extraction
* Case summarization using NLP
* TF-IDF, TextRank, cosine similarity
* Speech-to-text integration
* Streamlit dashboard + Flask backend

## 🛠 Tech Stack

* Python
* Flask
* Streamlit
* NLP libraries

## 📂 Project Structure

* `backend/` → Flask API
* `frontend/` → Streamlit UI

## ▶️ How to Run

```bash
cd backend
pip install -r requirements.txt
python app.py
```

```bash
cd frontend
streamlit run app.py
```
